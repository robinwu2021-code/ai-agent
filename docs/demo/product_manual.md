## 一、产品概述

智星AI营销平台是由星辰科技有限公司研发的一站式智能营销解决方案。平台依托大语言模型与多模态AI技术，帮助企业在微信公众号、抖音、微博、淘宝等主流渠道实现营销内容的自动化生成、精准投放与效果追踪，显著降低营销人力成本，提升ROI。

---

## 二、核心功能模块

### 2.1 多渠道营销管理

支持统一工作台管理微信公众号、抖音企业号、微博蓝V、淘宝店铺等渠道。用户可在单一界面完成内容发布、排期管理与互动回复，无需多平台切换。

**操作步骤：**
1. 进入"渠道管理"菜单，点击"绑定渠道"。
2. 选择目标平台，按提示完成OAuth授权。
3. 授权成功后，渠道状态显示为"已连接"。
4. 在"发布中心"新建内容，勾选目标渠道后点击"一键发布"。

### 2.2 AI内容生成

基于GPT级大语言模型，输入产品关键词、目标人群与营销目的，平台可在10秒内自动生成标题、正文、标签及配图建议，支持多风格切换（正式、活泼、促销、故事化）。

**操作步骤：**
1. 进入"AI创作"，选择内容类型（图文 / 短视频脚本 / 直播话术）。
2. 填写品牌名称、产品卖点（最多5条）及目标受众描述。
3. 点击"立即生成"，等待AI输出草稿。
4. 对生成内容进行编辑润色，点击"保存至内容库"。

### 2.3 用户画像分析

通过整合多渠道行为数据与CRM数据，构建360°用户画像，支持按消费偏好、活跃时段、地域分布、互动行为等多维度进行用户分群，为精准推送提供依据。

### 2.4 ROI分析与报表

实时追踪每次营销活动的曝光量、点击率、转化率及获客成本（CAC），支持自定义报表维度与数据导出（CSV / Excel / PDF格式）。

## 七、常见问题解答（FAQ）

**Q1：绑定微信公众号后，为何发布内容显示"授权过期"？**
A：微信公众平台的OAuth授权有效期为90天。当授权过期时，请进入"渠道管理"→"微信公众号"→"重新授权"，按提示完成重新绑定。建议开启"自动续期提醒"功能，系统将在到期前7天发送邮件通知。

**Q2：AI内容生成的结果质量不理想，如何改善？**
A：内容质量与输入的提示词密切相关。建议：（1）在"产品卖点"中填写3-5条具体特性，避免过于笼统；（2）在"目标受众"中描述人群年龄、职业、痛点；（3）选择与品牌调性匹配的风格模板。此外，可在"品牌设置"中上传品牌语料，系统将对模型进行微调，提升生成相关性。

**Q3：专业版的API接入如何使用？**
A：在"账户设置"→"API管理"中生成API Key。接口文档详见 https://docs.starlight-ai.com/api。每次调用需在请求头中携带`Authorization: Bearer <your_api_key>`。专业版每月API调用配额为50万次，超出后按¥0.01/次计费。

**Q4：用户画像数据从哪里来？如何导入自有CRM数据？**
A：平台自动采集各绑定渠道的行为数据（阅读、点赞、转发、购买等）。如需导入自有CRM数据，请进入"数据中心"→"数据导入"，下载CSV模板，按格式填写后上传。系统支持每次最多导入100万条用户记录，处理时间约30分钟。

**Q5：如何查看某次营销活动的ROI报告？**
A：进入"数据报表"→"活动分析"，在下拉菜单中选择目标活动，设置统计时间范围，点击"生成报告"。报告包含曝光量、点击率、转化率、获客成本及ROI总览。支持导出为PDF或Excel格式，方便对外汇报。
